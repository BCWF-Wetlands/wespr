# Retrieve indicator scores from a `wesp_site` object as a data.frame.

Retrieve indicator scores from a `wesp_site` object as a data.frame.

## Usage

``` r
get_indicator_scores(site, ...)
```

## Arguments

- site:

  A `wesp_site` object, having been updated with
  [`calc_indicators()`](https://bcwf-wetlands.github.io/wespr/reference/calc_indicators.md)

- ...:

  ignored for now

## Value

A data.frame of indicator scores
