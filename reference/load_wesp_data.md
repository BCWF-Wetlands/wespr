# Load the data containing the responses to WESP form

This file is usually an output from the R script at
https://github.com/BCWF-Wetlands/WESP_Calculator, and includes answers
to Office (OF), Field (F), and Stressor (S) questions.

## Usage

``` r
load_wesp_data(path)
```

## Arguments

- path:

  path to csv

## Value

a `data.frame` of the responses
