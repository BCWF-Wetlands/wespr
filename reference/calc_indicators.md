# Calculate Indicators for WESP assessment site

Calculate all Function and Benefit indicators for a site.

## Usage

``` r
calc_indicators(site)
```

## Arguments

- site:

  An object of type `wesp_site`, as created with
  [`as.wesp_site()`](https://bcwf-wetlands.github.io/wespr/reference/as.wesp_site.md).

## Value

`wesp_site` object with indicators calculated.
